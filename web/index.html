Connect your NES Advantage

Put the controller into config mode, then pair over Bluetooth to test every button live, tune how it behaves, and update firmware. Settings save to the device.

  1. 1On the stick, hold A + B + Select for 5 seconds. The blue LED blinks while it waits.
  2. 2Click Connect and pick NES Advantage Config.
Use Chrome or Edge on desktop or Android. On iPhone, Safari can't do Web Bluetooth, so use a Web BLE browser like Bluefy or BLE-Link. To leave config mode without a browser, hold Start ~3 s on the stick.
