# Jack plug

A 3D-printable plug that fills the original cable hole around the DC barrel jack. Print it in
transparent PLA so the status LEDs show through.

| File | Contents |
|---|---|
| `bt-nes-advantage-jack-plug.stl` | Print-ready mesh |
| `bt-nes-advantage-jack-plug.FCStd` | FreeCAD source |

The jack fits the CUI PJ-040 footprint in the same hole as the original NES Advantage cable. See
[`../../docs/HARDWARE.md`](../../docs/HARDWARE.md) for fit and assembly.
